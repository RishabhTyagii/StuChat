# StuChat ЁЯЪА

StuChat is a real-time chat application built with:

- Django + Django REST Framework
- Django Channels (WebSockets)
- JWT Authentication
- Next.js (App Router)
- P2P-ready architecture (WebRTC planned)

## Features
- Real-time 1-to-1 chat
- Typing indicator
- Online / Offline status
- Read receipts
- User search
- Profile pictures
- Inbox with last message preview

## Tech Stack
- Backend: Django, DRF, Channels
- Frontend: Next.js
- Database: SQLite / PostgreSQL
- Realtime: WebSockets
